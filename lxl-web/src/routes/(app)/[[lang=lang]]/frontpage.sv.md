# Välkommen till öppen beta!

Här kan du följa utvecklingen av nya libris.kb.se, prova nya funktioner och komma med feedback. Arbetet sker iterativt så förbättringar och nya funktioner läggs till allt eftersom. [Nyheter om librisutveckling](https://www.kb.se/for-bibliotekssektorn/nytt-fran-kb.html?sv.target=12.5b5507eb18cedffd1a434463&sv.12.5b5507eb18cedffd1a434463.route=/news&keywords=Libris&page=1) publiceras på kb.se.

I Libris hittar du information om vad som finns att få tillgång till på svenska bibliotek. [Initiativets](/about) slutgiltiga mål är att ersätta dagens söktjänst och ta steg vidare inom den tekniska utvecklingen för bibliotekssektorn.

_Att notera: Betaversionen innehåller en statisk kopia av Libris-katalogen som uppdateras periodiskt, det innebär att du inte nödvändigvis kan jämföra sökresultat mellan tjänsterna, även om det ofta går._
